# react-redux-boilerplate

This project is a suggested React projects structure.

It use the following technologies:
 - React 15
 - Redux 5
 - Webapack 1
 - Immutable 3
 - Babel 6

## Start it with
```
  npm install && npm start
```

## Build and distribuite it with
```
npm run build
```
